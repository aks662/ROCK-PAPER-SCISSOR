# 🪨 Rock Paper Scissors Game (React)

A simple and interactive **Rock Paper Scissors** game built using **React.js**.
This project demonstrates core React concepts like state management, event handling, and dynamic UI updates.

---

## 🚀 Features

* 🎮 Play Rock, Paper, Scissors against the computer
* 🤖 Random computer move generation
* 🧠 Winner logic implementation
* 🔥 Win streak tracking
* 📜 Match history tracking
* 🔄 Reset game functionality
* 🎨 Clean and responsive UI with custom CSS

---

## 🛠️ Tech Stack

* **React.js** (Functional Components & Hooks)
* **JavaScript (ES6+)**
* **CSS3**

## ⚙️ How It Works

1. User selects a move (Rock 🪨, Paper 📝, Scissor ✂️)
2. Computer randomly generates a move
3. Game logic determines the winner:

   * Rock beats Scissor
   * Scissor beats Paper
   * Paper beats Rock
4. Result, streak, and history are updated dynamically

---

## 🧩 Key Concepts Used

* `useState` for managing game state
* Event handling (`onClick`)
* Conditional rendering
* Array state updates (history tracking)
* Component re-rendering

## ▶️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/rock-paper-scissors.git
```

### 2. Navigate to project folder

```
cd rock-paper-scissors
```

### 3. Install dependencies

```
npm install
```

### 4. Run the app

```
npm run dev
```
